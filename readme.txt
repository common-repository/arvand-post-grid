Arvand Post Grid is a post grid plugin for displaying and styling your posts.

== Description ==
With this plugin you can greatly change the appearance of your site posts and have a flexible blog.
You can also use this grid post in search, categories, archives page, etc.
This plugin makes it as easy as possible for you to change and style home page posts.

= features =
* custom post types support
* customize posts wrap
* customize post image
* customize post avatar
* customize post title
* customize post excerpt
* customize post read more link
* customize post footer
* customize pagination
* change content elements order (title, excerpt, etc.)

== Installation ==
1. download and install the Arvand Post grid
2. activate the plugin
3. use apg_posts shortcode as described on information plugin page

== Frequently Asked Questions ==

= How to use wp arvand post grid in theme? =
You can use the following shortcode to use and display posts on your template pages: apg_posts
